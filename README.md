# UI Calculator JS

A clean and interactive UI calculator built using HTML, CSS, and JavaScript.  
This project includes a modern interface with number buttons, operators, a real-time display, and smooth hover effects.

## Features
- Modern UI with button grid
- Supports +, -, *, /
- Clear button (C)
- Real-time display updates
- Responsive layout
- Works in all browsers

## How to Use
1. Open `index.html` in any browser.
2. Click number and operator buttons.
3. Press "=" to calculate.
4. Press "C" to clear the screen.
